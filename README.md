# 해수 수질 종합 분석 대시보드 (K-Ocean WQI Dashboard)

본 프로젝트는 전국 해역 및 항만별 해수 수질 지수(WQI)를 분석하고 시각화하는 웹 기반 대시보드입니다. Python의 강력한 웹 프레임워크와 R의 정교한 통계/시각화 기능을 결합한 하이브리드 아키텍처로 구현되었습니다.

## 🛠️ 1. 사용 기술 및 구현 스택 (Tech Stack)
*   **Web Framework & UI**: Python (Streamlit)
*   **Data Processing**: Python (Pandas, Openpyxl)
*   **Interactive Visualization**: Python (Plotly Express)
*   **Static High-Res Visualization**: R (Base Graphics, `cairo` engine)
*   **Inter-process Communication (IPC)**: Python `subprocess` (Python과 R 간의 임시 CSV 파일 기반 데이터 통신 및 인코딩 제어)

## ⚖️ 2. Python과 R의 비중 및 역할 (비중 비율 = 약 70 : 30)

이 프로젝트는 **Python이 전체 시스템을 지휘(Controller)**하고, **R이 심층 분석 및 고해상도 시각화를 보조하는(Backend Engine)** 구조를 가지고 있습니다.

### 🐍 Python의 역할 (비중: 약 70%)
*   **웹 UI 구성**: Streamlit을 이용한 대시보드 화면, 파일 업로드 창, 탭(Tab) 메뉴, 필터링 사이드바 제공
*   **데이터 전처리 (ETL)**: 업로드된 원시 엑셀 데이터 로드, 병합 셀로 인한 결측치(NaN, None 등) 자동 복원(`ffill` 활용), '서해/동해/남해' 키워드 추론 및 문자열 정규화
*   **인터랙티브 시각화**: Plotly를 이용해 웹상에서 마우스를 올리면 상세 수치가 뜨는 반응형 웹 차트 생성
*   **프로세스 제어**: R 스크립트를 백그라운드에서 실행하며, 윈도우 한글 인코딩(CP949 ↔ UTF-8) 충돌을 방지하는 브릿지 역할 수행

### 📊 R의 역할 (비중: 약 30%)
*   **수질 지수(WQI) 통계 처리**: Python에서 1차 정제된 데이터를 넘겨받아 항만별 평균 WQI를 집계(`aggregate`) 및 점수별 수질 등급(1~4등급) 컬러링 분류
*   **고화질 전문 시각화 생성**: R의 Base Plot 기능과 안티앨리어싱 엔진(`cairo`)을 활용하여, 학술지나 보고서에 바로 쓸 수 있는 최고 화질(Retina 해상도)의 정적 막대그래프(PNG)를 동적으로 그려내어 파이썬에게 반환

## 💡 3. 핵심 포함 기능 (Key Features)

1. **다중 엑셀 파일 동시 분석 기능**
   * 한 번에 여러 개의 수질 데이터 엑셀 파일을 업로드할 수 있으며, 파일마다 독립된 탭(Tab)을 생성하여 병렬로 결과를 확인할 수 있습니다.
2. **스마트 데이터 결측치 보정 시스템**
   * 엑셀에서 자주 발생하는 병합 셀 구조를 완벽히 인식합니다. 빈칸이나 'None' 텍스트를 감지하여 바로 위 데이터(예: '남해')를 복사해 빈칸을 꽉 채워주고 필터링이 가능하도록 만듭니다.
3. **다차원 필터링 검색 시스템**
   * 좌측 사이드바를 통해 해역별(동해, 남해 등), 항만별, 수질 등급별(1~4등급), 오름차순/내림차순 등 데이터를 입맛에 맞게 즉각적으로 필터링할 수 있습니다.
4. **반응형 웹 대시보드 (Python 탭)**
   * 조건에 맞게 필터링된 데이터를 실시간 표(Dataframe) 형태로 보여주며, 평균 WQI와 데이터 건수 등 핵심 지표(Metric)를 제공합니다.
5. **초고화질 통계 시각화 (R 탭)**
   * 데이터 개수(n)에 따라 그래프의 비율과 폰트 크기를 스스로 조절하며, 픽셀 깨짐이 전혀 없는 고해상도 그래프를 렌더링합니다. 범례와 수치 겹침을 방지하는 스마트 레이아웃이 적용되어 있습니다.
6. **분석 결과 원클릭 내보내기 (Export)**
   * 결측치가 보정되고 정제가 완료된 최종 데이터셋을 언제든지 CSV 파일로 즉시 다운로드할 수 있는 버튼을 지원합니다.
